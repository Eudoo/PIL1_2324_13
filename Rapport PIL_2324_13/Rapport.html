Description de la Conception:

Notre application comprend les fonctionnalités ci-après:

  • Inscription : L'utilisateur peut s'inscrire en renseignant des informations de base (nom, prénom, âge, adresse), des photos, intérêts et une biographie et créer un pseudo
  • Connexion : Connexion à l'aide d'un pseudo et d'un mot de passe
  • Gestion de comptes: Après inscription et connexion, l'utilisateur a la possibilité de modifier quelques informations de son profil
  • Messagerie : L'utilisateur peut envoyer et recevoir des messages
  • Système de recommandations: L'utilisateur peut recevoir des recommandations de contacts, filtrées par age, intérêts et sexe
  • Mot de passe oublié : L'utilisateur peut demander un nouveau mot de passe en cas d'oubli
  • Système de notifications : L'utilisateur peut recevoir des notifications de messages des autres utilisateurs
  • Système de recherches : L'utilisateur peut filtrer les contacts en effectuant des recherches de pseudos ou en précisant le sexe et/ou les intérêts recherchés
  • Déconnexion : L'utilisateur peut se déconnecter de son compte
  • Interface utilisateur (UI) : Un design attrayant
  • Expérience utilisateur (UX) : Navigation fluide et réactive, c'est-à-dire, adaptée à tous types d'écrans
  • Base de données : Une base de données qui stocke toutes les données et actions des utilisateurs
  • Dashboard administrateur : L'administrateur peut créer, modifier et supprimer des profils

Instructions de Déploiement:

Pour déployer et exécuter notre application de rencontre, suivez les étapes ci-dessous :

Installez Django

  • Assurez-vous d'avoir le language Python installé sur votre machine. Vous pouvez télécharger python ici
  • Ouvrez votre terminal et exécutez la commande suivante pour installer Django à l'aide de l'outil de gestion de packages pip :

    pip install Django
  • Si vous n'avez pas pip installé, utilisez plutôt la commande suivante pour installer Django :

    python -m pip install Django

    Attendez que l'installation se termine. Une fois terminée, vous devriez avoir Django installé sur votre machine.
  • Pour vérifier si Django est installé correctement, exécutez la commande suivante dans votre terminal :

    django-admin --version

    Cela affichera la version de Django installée
  • Maintenant que vous avez Django installé, vous pouvez continuer à travailler sur le projet integrateur L1 IFRI PIL1_2324_13 .

Lancez l'application sur votre machine

Pour récuperer et lancer l'application sur votre machine, assurez vous d'abord d'avoir python et django installé sur votre machine. Ensuite procéder aux étapes suivantes :

  • git clone https://github.com/Eudoo/PIL1_2324_13.git
  • cd PIL1_2324_13
  • python manage.py migrate
  • python manage.py runserver

L'application va démarrer sur l'addresse http://localhost:8000 ou sur http://127.0.0.1:8000 :Lancez cette addresse dans le navigateur.
L'application est maintenant lancé !

Assurez-vous de suivre toutes les étapes et de configurer correctement votre environnement avant de déployer l'application. Si vous rencontrez des problèmes lors du déploiement, consultez notre manuel d'utilisation plus d'explication.

Manuel d'Utilisation:

Ce manuel d'utilisation fournit des instructions détaillées sur les fonctionnalités spécifiques réservées aux utilisateurs de Love Alarm afin de les aider à utiliser l'application.

Utilisateur:

- Sur la page d'accueil du site, l'utilisateur peut s'inscrire en suivant le formulaire d'inscription puis se connecter .
- Un utilisateur, si il a déjà un compte, peut se connecter en utilisant son pseudo et son mot de passe.
- Une fois sur la page de bienvenue, il peut:
  • Rechercher des utilisateurs avec leur pseudo ou en fonction de la ville, du sexe et/ou des intérêts.
  • Voir et modifier son profil dans le menu "Profil".
  • Envoyer et recevoir des messages pour communiquer dans la section "Messages".
  • Parcourir les profils suggérés et utiliser la fonction "J'aime" pour signaler son intérêt pour un partenaire potentiel dans le menu "Recommandations". Il peut également utiliser la fonction "Envoyer un message" pour une prise de contact.
  • Voir ses notifications de messages dans le menu "Notifications".
  • Se déconnecter en appuyant sur "Déconnexion".

Administrateurs:

Les administrateurs du site peuvent aussi se connecter en accédant à l'adresse http://localhost:8000/admin dans le navigateur puis en renseignant identifiant et mot de passe administrateurs.
Ils ont la capacité d'ajouter, modifier et supprimer des utilisateurs.
